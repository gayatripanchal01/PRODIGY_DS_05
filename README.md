# 🚦 Traffic Accident Analysis & Hotspot Detection

## 📌 Project Overview
This project analyzes real-world traffic accident data to identify patterns based on:

- Time of Day
- Weather Conditions
- Day of Week

It also detects accident-prone zones using heatmap visualization.

---

## 🎯 Objective
The goal of this project is to:

✔ Identify accident trends  
✔ Understand environmental factors  
✔ Detect accident hotspots  

---

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Folium

---

## 📂 Dataset
US Accidents Dataset (Kaggle)

---

## ⚙️ Project Workflow

### 1️⃣ Data Preprocessing
- Converted timestamps
- Removed missing values
- Extracted Hour & Day

### 2️⃣ Data Analysis
Analyzed accidents based on:

- Hour of the day
- Day of the week
- Weather conditions
- Severity level

### 3️⃣ Visualization
Generated:

- Hour-wise accident trends
- Day-wise accident distribution
- Weather impact analysis
- Severity insights

### 4️⃣ Hotspot Detection
Created an interactive heatmap showing accident-prone locations.

---

## 📊 Outputs Generated

Inside `outputs/` folder:

- Accidents by Hour Graph
- Accidents by Day Graph
- Weather Analysis Graph
- Severity Distribution
- Severity vs Time
- Severity by Day
- Accident Hotspot Map (HTML)

---

## 📈 Key Insights

- Accidents peak during rush hours.
- Weekdays show higher accident frequency.
- Adverse weather increases accident risk.
- Urban regions show dense accident clusters.

Traffic accidents are influenced by temporal and environmental factors.  
Hotspot detection can help authorities improve road safety and planning.
